"Hotel Booking System"

The Hotel Booking System is a simple web application that allows users to book hotel rooms online. Users can select different room types, enter booking details, and view room images. This project demonstrates basic frontend web development using HTML, CSS, and JavaScript.

Features:

Attractive hotel header with banner image
Display of different room types
User-friendly booking form
Responsive layout
Simple and clean UI design

Technologies Used:

HTML
CSS
JavaScript
